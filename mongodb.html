MongoDB Installation and Setup

Install MongoDB on MacOS

  1. First, make sure that you have Xcode and Homebrew installed before you get started.

  2. To install MongoDB on macOS, run the following commands:

    brew tap mongodb/brew
    brew install mongodb-community@4.2

    Note: The version number following the @ symbol may have changed. See the MongoDB tutorial on installing on macOS to make sure that you have the latest version to use with this command.

  3. Run the following command in your terminal to start up MongoDB. If this fails, it may be due to a problem with the path to Homebrew where MongoDB was installed. 

    brew services start mongodb-community@4.2

  4. Run the following command to see if MongoDB is running:

    ps aux | grep -v grep | grep mongod

    If you get an error, there may be a problem with your Mac's path.

    export PATH=$PATH:/usr/local/Cellar/mongodb-community@4.2/4.2.17/bin

    You may have to adapt for the version of MongoDB you're running and where Homebrew installed MongoDB. Add the line below to the startup script for your login shell.

  5. Initialize the Mongo shell by entering the following command:

    mongo

Install MongoDB Compass on macOS

Now that MongoDB is installed, we need to install MongoDB Compass. Compass is a GUI (graphical user interface) that allows the user to query and view a MongoDB database and use the embedded MongoDB shell.
